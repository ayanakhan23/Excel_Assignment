Excel Practical Assignment – End-to-End Excel Skills Case Study

Project Overview -
This repository contains a comprehensive Excel practical assignment designed to strengthen real-world Excel skills used in data analysis, reporting, and business decision-making.
The project covers end-to-end Excel functionality, starting from basic formulas to advanced topics like Pivot Tables, conditional aggregation, data cleaning, consolidation, and VBA Macros.
All tasks were completed entirely using Microsoft Excel, without using any external programming tools.
The dataset used for practice is the Sample Superstore Sales dataset, a widely used business dataset for sales and performance analysis.

Objective -
The primary objective of this assignment is to:
Build strong hands‑on Excel skills
Understand how Excel is used in real business scenarios
Perform data analysis, cleaning, formatting, and reporting efficiently
Develop confidence in using Excel for analytics, operations, and reporting roles
This assignment is mandatory and builds on the foundational Excel knowledge from Module 1.

Dataset - 
Dataset Name: Sample Superstore Sales
Source: Tableau Community
Type: Sales & Operations Dataset
Data Includes: -
Order dates
Product details
Sales, profit, and quantity
Customer and regional information
The dataset was used across multiple sections to simulate real‑world business use cases.

Assignment Coverage -
1. Basics of Excel -
Arithmetic functions: SUM, MAX, MIN, SMALL, LARGE, MEDIAN
Transpose vs TRANSPOSE() formula
COUNT, COUNTA, COUNTBLANK
Quick calculations using Status Bar
Year‑over‑Year growth & CAGR using POWER()

2. Formatting in Excel -
Cell formatting & currency formatting
Conditional formatting for performance analysis
Custom number formatting
Text formatting using Wrap Text, alignment & rotation
Efficient formatting using Format Painter

3. Sort & Filter Operations -
Sorting numbers, dates, and text (A–Z / Z–A)
Sorting by cell color
Multi‑level sorting for hierarchical analysis
Advanced sorting tricks for clean reporting

4. Working with Dates -
Date & timestamp handling
Duration calculation between two dates
DATEDIF, TODAY, DATE, YEAR, MONTH, DAY
Dynamic date‑based analysis

5. Logical Functions -
IF, AND, OR, IFERROR
Nested logical formulas
Logical tests on text, numbers, and error values
Business classification logic (High / Low, Pass / Fail)

6. Practical Data Cleaning -
Removing duplicates & error values
Text cleaning using UPPER, LOWER, PROPER, TRIM
String functions: LEFT, RIGHT, MID, LEN, VALUE
SEARCH vs FIND :
Go To (Special) for fast data cleaning
Text to Columns
CONCATENATE & & operator
Find & Replace with wildcards

7. Pivot Table (A–Z) -
Creating Pivot Tables from raw data
Grouping by date (month, quarter, year)
Handling blank values in Pivot Tables
Custom grouping & conditional formatting
Multi‑analysis using Pivot Tables

8. Lookup Functions -
VLOOKUP basics and advanced usage
Exact match vs approximate match
Error handling using IFERROR
VLOOKUP vs HLOOKUP
Named ranges with lookup formulas

9. Conditional Aggregation -
COUNTIF, SUMIF, AVERAGEIF
COUNTIFS & SUMIFS for multi‑criteria analysis
Business‑driven aggregation logic

10. Conditional Formatting -
Color‑based performance analysis
Color scales, data bars, and icon sets
Visual representation of trends and KPIs

11. Report Consolidation -
Consolidation across multiple sheets
CONSOLIDATE feature (SUM, AVERAGE, COUNT)
SUBTOTAL function with filters
Creating dynamic summary reports

12. Basics of VBA Macros -
Introduction to VBA Macros
Enabling Developer Tab
Recording, editing, and running macros
Running macros using buttons, shortcuts & events
Understanding VBA Editor workspace
Finding and customizing VBA code snippets
